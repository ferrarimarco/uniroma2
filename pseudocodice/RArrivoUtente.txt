* DESCRIZIONE PROCEDURA < Arrivo Utente >;
PROCEDURE Arrivo Utente;
BEGIN

	* GESTIONE JOB ENTRANTE

	* CREO NUOVO JOB
	< creo nuovo JOB J classe 1 >;
	< inizializzo J >;

	* CONTROLLO SE JOB NECESSITA DI HOST
	< creo nuovo HOST H per J >;

	* CREO STAMPANTE PER JOB
	< creo nuova STAMPANTE ST per J >;

	* INVIO A CPU
	< prossima unità NX(J) = CPU per J >;

	* USCITA JOB DA DISK
	IF < CPU libera >
		THEN
		< inserisco J in CPU >;
	ELSE
		< inserisco J in codaCpuClasse1 >;

	* LIBERO TERMINALE (non mi preoccupo della coda)
	< libera terminale associato a J >;

	* AGGIORNAMENTO CALENDARIO
	< prevedi durata Tcpu(J) del servizio cpu >;
	< aggiorno calendario tcpu := clock + Tcpu(J) >;

	

END
